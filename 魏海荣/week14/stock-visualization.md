# Stock Visualization Skill

## 描述

股票可视化与分析技能，提供周波动和日波动图表绘制，以及基于波动率的买卖时机建议。

## 核心功能

### 1. 波动率可视化
- **日波动图**: 绘制每日的涨跌幅柱状图
- **周波动图**: 叠加显示周级别波动趋势线
- **双周期对比**: 在同一图表中对比日/周波动，便于识别短期与中期趋势

### 2. 波动率计算
- 日波动率: `(今日收盘价 - 昨日收盘价) / 昨日收盘价 * 100%`
- 周波动率: `(本周收盘价 - 上周收盘价) / 上周收盘价 * 100%`
- 波动率排名: 与历史水平比较，判断当前波动高低

### 3. 买卖时机建议

基于波动率的分析逻辑:

| 波动状态 | 日波动 | 周波动 | 建议 |
|---------|--------|--------|------|
| 低波动平稳 | < 1% | < 2% | 观望，等待突破确认 |
| 中度波动 | 1-3% | 2-5% | 谨慎操作，适合定投 |
| 高波动 | 3-5% | 5-8% | 关注买卖点，适合短线 |
| 极端波动 | > 5% | > 8% | 谨慎追涨杀跌，考虑止盈止损 |

**买入信号**:
- 日波动率由低转高，且周波动向上 → 趋势启动，考虑买入
- 极端下跌后波动率收缩 → 可能是底部信号

**卖出信号**:
- 波动率创近期新高但价格未创新高 → 顶背离，考虑卖出
- 持续高波动后波动率开始下降 → 趋势可能反转

## 使用方法

```python
from stock_visualization import StockVisualizer

# 初始化
visualizer = StockVisualizer(stock_data)  # stock_data 需包含 date, open, high, low, close, volume

# 绘制波动图
visualizer.plot_volatility()

# 获取买卖建议
advice = visualizer.get_buy_sell_advice()
```

## 数据格式要求

输入数据需包含以下字段:
- `date`: 日期
- `open`, `high`, `low`, `close`: OHLC价格
- `volume`: 成交量(可选)

## 输出

1. **可视化图表**: 日/周波动叠加图，标注关键波动点
2. **分析报告**: 当前波动状态、趋势判断、具体买卖建议
