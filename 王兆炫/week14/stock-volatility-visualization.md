# 股票日/周波动可视化与时机提示

## 适用范围

在用户需要：**同一张图**展示**日波动**与**周波动**，并基于波动**相对大小**给出**买入/卖出时机倾向性建议**（非投资建议）时遵循本技能。

## 合规与边界

- 所有输出必须标注：**不构成投资建议**；仅作技术分析与可视化教学。
- 不承诺收益；不替代持牌顾问意见。
- 若缺少数据许可或 API 密钥，说明限制并改用用户上传的 CSV（需含日期与收盘价，可选高低价）。

## 数据与定义（术语统一）

| 序列 | 推荐定义 | 说明 |
|------|----------|------|
| 日波动 | 对**日收益率**（优先对数收益 `log(close/close.shift(1))`）做**滚动标准差**，窗口默认 **20 个交易日**，可再年化：`日波动_年化 = 日滚动std * sqrt(252)` | 与常见「历史波动率」一致 |
| 周波动 | 将价格**按周聚合**（如周五收盘），算**周收益率**序列，再对其做**滚动标准差**，窗口默认 **12 周**，年化：`周波动_年化 = 周滚动std * sqrt(52)` | 低频噪声更少 |
| 对齐到同一横轴 | 将**周波动序列**按周末日期对齐后，**前向填充**到日频索引，与日波动画在同一张图 | 保证「一图两序列」可比 |

可选增强（若用户提供 OHLC）：日/周可用 **Garman–Klass** 或 **Parkinson** 估计器替代纯收盘收益波动；在技能中二选一并在图例注明。

## 可视化规范

1. **单图双序列**：左轴或单轴两条线——**日波动（年化）**、**周波动（年化）**；图例、标题、数据来源、起止日期写清。
2. **辅助参考**：可加**日波动 / 周波动**比值曲线（次轴或子图），用于「相对大小」判断。
3. **配色**：两序列对比度足够；关键分位线（如滚动窗口内的 20%/80% 分位）可用浅色虚线可选展示。
4. **交付物**：优先给出可复现脚本（Python：`pandas` + `matplotlib` 或 `plotly`）。若用户需要 Cursor 内交互展示，可用 Canvas（见团队 canvas 技能）承载图表，但**不强制**。

## 基于「大小」的买卖时机**倾向**规则（启发式）

以下规则仅描述**波动结构常见解读**，须配合上文免责声明；应用时写明触发条件与反例。

1. **双低压缩（日波动与周波动均处于各自滚动分位的低位，如 <30%）**  
   - **含义**：波动能量积聚，变盘概率上升。  
   - **倾向提示**：更常讨论「等待方向确认后的**右侧**参与」或「网格/定投的**节奏**」而非盲目抄底。  
   - **卖出侧**：若已盈利且波动极低后**放量突破失败**，可提示注意**利润保护**（非指令性）。

2. **周高、日低（周波动高、日波动相对不高）**  
   - **含义**：中长期不确定性大，但短线尚未剧烈抖动。  
   - **倾向提示**：趋势或事件驱动可能主导；新多单宜**轻仓/宽止损**；已有仓位可强调**风险预算**。

3. **日高、周低（日波动骤升、周波动仍不高）**  
   - **含义**：短线情绪或噪声放大。  
   - **倾向提示**：警惕**假突破**与**过度交易**；卖方语境下可讨论**短线超买后的减仓节奏**（定性）。

4. **双高共振（日与周波动均处于高分位，如 >70%）**  
   - **含义**：风险定价剧烈。  
   - **倾向提示**：趋势策略强调**跟踪止损**；均值回归策略强调**避免接飞刀**；默认**不**给出「全仓买卖」类措辞。

5. **比值极端**  
   - **日/周 比值显著高于历史中枢**：短线扰动占主导 → 提示**勿过度解读单日**、注意**费用与滑点**。  
   - **日/周 比值显著低于中枢**：价格变动「慢而稳」→ 可讨论**趋势延续**与**波动率卖出策略**风险（卖方需专业门槛）。

每条建议必须输出：**触发指标与阈值**、**失效条件**（如跌破关键价位/波动模型假设不成立）。

## 工作流（代理执行顺序）

1. 确认标的、区间、复权方式（前复权/不复权）、时区与交易日历。  
2. 拉取或读取 OHLCV；清洗缺失与停牌。  
3. 按上表计算日波动、周波动并对齐。  
4. 绘制单图；导出 PNG/SVG 或 HTML（plotly）。  
5. 计算当前与近 N 期的分位与比值；套用上节规则生成**分条、带条件**的文字说明。  
6. 结尾重申：非投资建议。
