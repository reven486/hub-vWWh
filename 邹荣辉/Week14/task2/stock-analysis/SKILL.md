# 股票分析与可视化技能

## 何时使用
- 用户提到具体股票代码并希望可视化其走势
- 用户想看某只股票的波动率（日波动 / 周波动）
- 用户询问某只股票的买入/卖出时机建议

## 用法

```bash
python scripts/analyze.py <股票代码> [--start YYYYMMDD] [--end YYYYMMDD] [--output PATH.png]
```

参数：
- `<股票代码>`：
  - A 股：6 位数字（如 `600519` 贵州茅台、`000001` 平安银行）→ 自动走 akshare
  - 美股/港股：字母 ticker（如 `AAPL`、`MSFT`、`0700.HK`）→ 自动走 yfinance
- `--start`：开始日期（默认一年前）
- `--end`：结束日期（默认今天）
- `--output`：图表输出路径（默认 `stock_analysis.png`）

示例：
```bash
python scripts/analyze.py 600519
python scripts/analyze.py AAPL --start 20230101 --end 20241231
python scripts/analyze.py 0700.HK --output tencent.png
```

## 输出

### 控制台
- 数据条数与时间范围
- 最佳买入日期 / 价格 / 当日波动率
- 最佳卖出日期 / 价格 / 当日波动率
- 潜在收益率（%）
- 日波动 / 周波动 的均值与最大值

### 图表（PNG，单张）
- **左 Y 轴**：收盘价线 + 候选买卖点小三角 + 最佳买卖点高亮标注
- **右 Y 轴**：
  - 日波动率（灰色柱状图，半透明）
  - 周波动率（橙色折线）
- **波动率定义**：`(最高 − 最低) / 收盘 × 100%`（周波动按周聚合）

## 推荐逻辑（"基于大小"）

1. **找出局部极值**：以 ±10 天为窗口，找到局部价格最低点和最高点
2. **波动阈值过滤**：候选点的当日波动率必须 ≥ 全期日波动率的 75% 分位数（即"波动够大"）
3. **最佳买入**：候选买点中价格最低的
4. **最佳卖出**：必须晚于最佳买入，候选卖点中价格最高的
5. **退化策略**：若波动过滤后无候选，回退到区间全局最低/最高

> 设计意图：高波动往往出现在情绪极端的拐点（恐慌底 / 狂热顶），因此"波动大 + 局部极值"是合理的择时信号。
