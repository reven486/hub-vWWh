### mineru和 pdfplumber的效果差异
Mineru:
这是一款拥有12亿参数的文档解析视觉-语言模型，在实现顶尖识别准确率的同时，还具备出色的计算效率。
它采用了从粗到细的两阶段解析策略，将全局布局分析与局部内容识别分离开来。
在第一阶段，模型对下采样图像进行高效的布局分析，以识别结构元素，从而避免处理高分辨率输入带来的计算开销。
在第二阶段，在全局布局的引导下，模型对从原始图像中提取的原生分辨率裁剪区域进行针对性的内容识别，保留密集文本、复杂公式和表格中的细粒度细节。

- 有视觉模型
- 能自动处理双栏 / 多栏、分栏重排、阅读顺序校正；支持中英混排、特殊符号、脚注 / 尾注、公式 / 表格。
- 支持扫描 PDF、图片 PDF、手写体（部分）；自动区分文本 / 公式 / 表格区域，OCR 后结构化输出
- 支持格式化输出，MarkDown/Json,可直接用于知识库

pdfplumber:
是一款纯 Python 编写、轻量级、开源的 PDF 文本与表格提取库，基于 pdfminer.six 开发
专门用于精准提取原生 PDF 中的文字、表格、坐标信息，无 AI、无 OCR、无公式识别，
主打稳定、轻量、可控

- 无模型，无依赖，速度更快
- 单栏正常，双栏必乱，复杂布局（如环绕图文）顺序错乱；无多语言优化
- 仅支持简单边框表格，完全不支持公式，公式区域直接丢弃或乱码；无任何公式处理能力
- 无 OCR 能力，仅处理原生文本 PDF
- 无结构化输出，仅输出存文本
